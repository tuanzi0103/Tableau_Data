# Student Academic Performance Dashboard

This project presents an interactive **Student Academic Performance Dashboard** developed in Tableau. The dashboard is designed to provide actionable insights into student performance by visualizing learning patterns, academic progress, and the underlying factors influencing final grades.

## Live Demo
Explore the interactive dashboard here: 
[Tableau Public - Student Academic Performance Dashboard](https://public.tableau.com/app/profile/ruby.gao/viz/StudentAcademicPerformanceDashboard_17840913151540/Dashboard1?publish=yes)


##  Dashboard Overview
The dashboard utilizes data to help educators and analysts identify trends and understand student achievement dynamics. Key visualizations include:

*   **KPI Cards**: High-level metrics including average final grade and student pass/fail status.
*   **Grade Trend**: A longitudinal view of student performance across G1, G2, and G3 periods.
*   **Core Metrics**:
    *   Gender Distribution analysis.
    *   Absence vs. Grade correlation.
    *   Study Time impact on academic outcomes.
    *   Comparative analysis of school performance.


##  Proposed Enhancement Features
To evolve the dashboard from descriptive to prescriptive analytics, I have designed three enhancement features:

### 1. Student Learning Recommendation
Provides personalized learning interventions based on performance indicators such as study time, failures, and absences. Recommendations include:
*   Increase Study Hours
*   Attendance Support
*   Academic Tutoring
*   Maintain Performance

### 2. Family Background Analysis
Explores external factors by correlating variables like parental education level (Medu/Fedu) and family support (famsup) with academic outcomes to provide a holistic view of student performance.

### 3. Learning Behavior Analysis
Investigates the relationship between specific learning habits, previous failures, and final academic achievement to identify behavior patterns that contribute to success.
